Multiple HR Payslip Confirm

Overview

This module will allow enable you to confirm multiple HR Payslip.

How It Works

In payslip tree view, select the payslips yu want to confirm and from action select Confirm Payslips.
If you selected any payslip not in draft state, the module will ignore them.
