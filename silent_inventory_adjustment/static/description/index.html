Silent Inventory Adjustment

  • Silent Inventory Adjustment helps to Export/Import product stock using CSV File.
  • This functionality supports both Export and Import features.
  • It allows us to export the product list from inventory adjustment and also Imports the CSV file again in the Inventory along with the changes.
  • Prepare a CSV file with the product code and then it can be Exported/Imported.

Installation Procedure

For Installation - Go to apps -> Search Silent Inventory Adjustment -> Install.

Procedure to use

After Installation - Go to Inventory -> Click on Inventory Adjustments.

Create Inventory Reference

Click on Create.

Fill the details -> Click on Save -> Click on start Inventory.

Wizard to Export the CSV file of Inventory Adjustments

Click on Action -> Click on -> Export.

Wizard to Export the CSV file of Inventory Adjustments

After making the necessary changes, Import the CSV file again to the Inventory Adjustments.

Wizard to Import the CSV file of Inventory Adjustments

Click on Action -> Click on -> Import.

Click on Upload your file.

Select the file from File Manager -> Click on Import.
The file is Imported to the Inventory Adjustments along with all the necessary changes.
